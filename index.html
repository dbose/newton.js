newton.js

A much-needed wrapper over Box2D for RaphaelJS. Newton.JS is not supposed to be a complete game engine, although the game-loop, behaviors has creeped in the code. This has been extracted from Mixow

Inspiration

This project has been inspired from RaphaelJS creator Dmitry's awesome presentation at Sydney, 2011 - http://whatdoyouknow.webdirections.org/videos/newton-js-box2d-raphael

Dependencies

  • Box2DWeb - https://code.google.com/p/box2dweb/
  • RaphaelJS - https://github.com/DmitryBaranovskiy/raphael/
